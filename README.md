# Toilet Survivor

A pixel-art top-down mobile survival shooter built with Flutter and Flame.

## Audio Notes

Use free game-audio sources first:

- Kenney: free CC0 game UI/retro sound packs.
- Sonniss GDC Game Audio Bundle: free royalty-free game audio bundle.
- Pixabay Sound Effects: free sound effects and music loops.
- Freesound: use the CC0 filter when possible.

Avoid audio with `NonCommercial` licenses because this game uses ads. If an
asset requires attribution, save the source URL and author name before adding it
to the project.

Suggested search keywords:

- UI: `ui click`, `button click`, `menu select`, `pause sound`, `start button`,
  `restart button`, `game ui blip`, `retro ui beep`.
- Player shooting: `retro gun shot`, `arcade shoot`, `laser shot`,
  `pistol pop`, `cartoon gun`, `bullet fire`, `muzzle pop`.
- Bullet hit: `hit marker`, `impact pop`, `wet impact`, `splat hit`,
  `cartoon impact`, `slime hit`, `soft body impact`.
- Enemy death: `poop splat`, `mud splat`, `gross splat`, `slime splat`,
  `wet splat`, `cartoon splat`, `squelch`.
- Player damage: `player hurt`, `damage hit`, `body hit`, `short pain`,
  `impact thud`, `hurt beep`.
- Sludge puddle: `slime step`, `mud step`, `wet squelch`, `slime slide`,
  `slippery slide`, `goo step`.
- Stink/fart hazards: `fart`, `gas puff`, `poison cloud`, `stink cloud`,
  `toxic gas`, `gas burst`, `cartoon fart`.
- Snake enemy: `snake hiss`, `slither`, `creature hiss`, `small monster hiss`.
- Boss: `monster roar`, `gross monster`, `boss spawn`, `large creature`,
  `warning sting`, `monster gurgle`.
- Ambience: `sewer ambience`, `water drip`, `pipe drip`, `underground ambience`,
  `drain ambience`, `wet tunnel`, `dark sewer loop`.
- Music: `retro game loop`, `pixel game music`, `arcade survival loop`,
  `dark chiptune`, `boss music loop`, `game over jingle`.

Recommended MVP audio order:

1. UI click/start/restart.
2. Shoot, bullet hit, enemy death.
3. Player hurt and game over.
4. Gameplay music loop.
5. Boss spawn and stink/fart hazards.
6. Sewer ambience loop.
